# OUWIBO NFT Protocol

A premium digital asset infrastructure built on the **Base network** and optimized for the **Farcaster** social ecosystem.

## Core Features
- **Official Genesis Mint**: Access the primary protocol pass (ERC-1155).
- **Gasless Infrastructure**: Server-side sponsorship for seamless user onboarding.
- **Multi-Platform Integration**: Native support for Farcaster Mini-apps and Coinbase Base App.
- **Atlantis Design System**: Futuristic, mobile-first Web3 interface.

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Blockchain**: Base Mainnet (L2)
- **SDKs**: Thirdweb v5, Farcaster Frame SDK v2
- **Styling**: Tailwind CSS, Framer Motion

## Configuration
Requires environment variables for `THIRDWEB_SECRET_KEY`, `NEXT_PUBLIC_CONTRACT_ADDRESS`, and `BACKEND_PRIVATE_KEY` (for gasless transactions).
